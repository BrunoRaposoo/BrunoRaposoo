## 🔥 Projeto em destaque

### 🏥 Clínica SaaS

Sistema **multi-tenant completo** para gestão de clínicas e consultórios.

**Diferenciais de engenharia:**
- Arquitetura multi-tenant com isolamento por organização  
- RBAC com permissões granulares  
- Agenda inteligente com validação de conflitos  
- Módulo financeiro com automação de status  
- Dashboard com KPIs e análise temporal  
- Processamento assíncrono com filas (Redis + BullMQ)  

**Stack:**
Next.js · NestJS · Prisma · PostgreSQL · Redis  

🔗 **Repositório:**  
https://github.com/BrunoRaposoo/clinica-saas
